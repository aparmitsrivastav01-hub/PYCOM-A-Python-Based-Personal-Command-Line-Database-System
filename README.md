PYCOM — A Python-Based Personal Command-Line Database System

PYCOM is a beginner-friendly Python project designed to interact with a MySQL database using CRUD (Create, Read, Update, Delete) operations. The system runs through a clean command-line interface and allows users to store, update, manage, and delete user records securely.

This project demonstrates:

Python × MySQL connectivity

Database authentication and query handling

Error handling and input validation

Terminal UI aesthetics with ASCII art banners

Modular and readable code structure

PYCOM is a great starter project for learning database programming fundamentals and understanding how Python interacts with external systems.
